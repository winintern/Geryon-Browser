# 🦅 Geryon Browser — v1.3 «Тёмный страж»

> **Geryon** — приватный браузер на движке Chromium (QtWebEngine), созданный для тех, кто ценит конфиденциальность, скорость и полный контроль над данными.  
> Эта версия — **самое масштабное обновление**: 10 новых фич, которые превращают браузер в полноценный инструмент для ежедневного серфинга.

## 🚀 Что нового в версии 1.3?

| № | Фича | Описание |
|---|------|----------|
| 1 | 💾 **Сохранение сессии** | Вкладки восстанавливаются при запуске — не теряй рабочие страницы. |
| 2 | 🌙 **Тёмный режим для сайтов** | Инвертирует цвета на любой странице, бережёт глаза. |
| 3 | 🔊 **Звуковые уведомления** | Завершение загрузки, обновления и ошибки теперь сопровождаются звуком. |
| 4 | 📦 **Импорт/экспорт профиля (ZIP)** | Переноси все настройки, закладки, историю и пароли одним кликом. |
| 5 | 🔍 **Автодополнение закладок** | В адресной строке появляются подсказки из сохранённых ссылок. |
| 6 | 🧮 **Калькулятор в адресной строке** | Введи `2+2*3` — и получи результат без перехода на сайт. |
| 7 | 📜 **Поиск по закладкам и истории (Ctrl+Shift+F)** | Находи нужное в один клик по всей коллекции. |
| 8 | 📅 **История закладок** | Сохраняется дата добавления, сортировка по времени. |
| 9 | 🔗 **Drag & Drop для ссылок** | Перетаскивай ссылки из адресной строки на рабочий стол или в закладки. |
| 10 | 🌐 **Выбор языка интерфейса** | Переключайся между русским и английским (перезапуск). |

## ✨ Уже знакомые фичи

- Вкладки, закладки, история, быстрые ссылки (Speed Dial).
- Менеджер загрузок с прогрессом (Ctrl+J).
- Встроенный менеджер паролей (AES-256 + мастер-пароль).
- Блокировщик рекламы (локальный список, обновляется ежедневно).
- Защита от fingerprinting (подмена User-Agent, отключение WebGL/Canvas).
- Режим «Только текст» и экономия трафика (отключение картинок/CSS/видео).
- Паника (Ctrl+Shift+X) — мгновенная очистка сессии.
- Тёмная тема интерфейса + кастомный акцентный цвет.
- Стартовая панель с погодой, курсами валют и часто посещаемыми.
- Автообновление через GitHub Releases.

## 📥 Установка

### Быстрая установка (рекомендуется)
1. Скачай установщик из раздела [Releases](../../releases) или по [прямой ссылке](https://github.com/winintern/Geryon-Browser/releases/download/v1.3/GeryonSetup.exe).
2. Запусти `GeryonSetup.exe` (требуются права администратора).
3. Выбери компоненты (ярлыки на рабочем столе/в меню Пуск) и дополнительные задачи (создание папки для загрузок).
4. После установки запусти Geryon из меню Пуск или с рабочего стола.

Для проверки, что установщик не повреждён и не подменён:

1. Скачайте `GeryonSetup.exe` ` из одного релиза.
2. Откройте PowerShell в папке с файлами.
3. Выполните команду:

```powershell
Get-FileHash GeryonSetup.exe -Algorithm SHA256 | Format-List
