# Geryon Browser

**Geryon** – приватный браузер на движке Chromium (QtWebEngine) с упором на безопасность и удобство.  
Закладки, история, быстрые ссылки, тёмная тема, режим «Только текст», менеджер загрузок с прогрессом и паника (Ctrl+Shift+X).

## ✨ Что нового в версии 1.2.1

- 🔄 **Автоматическая проверка обновлений** – при запуске браузер сам проверяет новую версию на GitHub и предлагает скачать установщик.
- 📊 **Стартовая панель с виджетами** – погода, курсы валют, последние закладки (открывается кнопкой на панели или как стартовая страница).
- 👥 **Профили пользователей** – создавайте несколько профилей (закладки, история, настройки хранятся отдельно) и переключайтесь между ними без потери данных.

## 🚀 Установка

1. Скачай установщик по [этой ссылке](https://github.com/winintern/Geryon-Browser/releases/download/v1.2.1/GeryonSetup.exe).
2. Запусти установщик (требуются права администратора).
3. Выбери компоненты (ярлыки на рабочем столе/в меню Пуск) и дополнительные задачи.
4. После установки запусти Geryon из меню Пуск или с рабочего стола.

## 🔧 Возможности

- **Вкладки** (Ctrl+T – новая, Ctrl+W – закрыть, Ctrl+Tab – переключение).
- **Адресная строка** с поиском (DuckDuckGo, Google, Bing, Яндекс – выбор в меню «Поиск»).
- **Закладки** (Ctrl+D – добавить, меню «Закладки» – просмотр).
- **История** посещений (меню «История» – показать/очистить).
- **Быстрые ссылки** (Speed Dial) на стартовой вкладке.
- **Стартовая панель** – погода, курсы валют, последние закладки (кнопка на панели).
- **Профили** – переключение между разными наборами данных (меню «Настройки» → вкладка «Профили»).
- **Автообновление** – проверка новых версий при запуске.
- **Режим «Только текст»** (отключает загрузку изображений, ускоряет на слабых ПК).
- **Тёмная тема** интерфейса (меню «Вид»).
- **Режим «Читалка»** (упрощает страницу до текста).
- **Менеджер загрузок** (Ctrl+J) – прогресс, отмена, открытие папки, история загрузок.
- **Паника** (Ctrl+Shift+X) – очищает текущую сессию и кэш.
- **Приватность:** все данные (история, куки, кэш) хранятся на диске, но вы можете очистить их в любой момент.

## ⚙️ Системные требования

- Windows 10 / 11 (64-bit)
- 2 GB ОЗУ (рекомендуется 4 GB)
- 200 MB свободного места

## 🔒 Права администратора

Для **установки** Geryon Browser в папку `Program Files` требуется запустить установщик с правами администратора.  
Это стандартное требование Windows для программ, которые устанавливаются системно.

После установки браузер **работает без прав администратора** и:
- не вносит изменения в реестр (если вы не выбрали опции ассоциации файлов или автозагрузки);
- не требует повышения привилегий при обычном запуске;
- не выполняет никаких системных действий, кроме чтения/записи в свою папку профиля (`%APPDATA%\Geryon`).

Можете смело пользоваться браузером с обычной учётной записью.
